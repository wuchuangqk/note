# 基础操作

## 快捷键

| 快捷键                | 功能                         |
| --------------------- | ---------------------------- |
| Alt + 左键 / 3 + 左键 | 旋转视角                     |
| Alt + 右键 / 2 + 左键 | 缩放视角                     |
| Alt + 中键 / 1 + 左键 | 平移视角                     |
| C                     | 把多边形转换成`可编辑多边形` |
| F1                    | 透视试图独占                 |
| F2                    | 顶视图独占                   |
| F3                    | 右视图独占                   |
| F4                    | 正视图独占                   |
| F5                    | 四视图并列展示               |
| T                     | 缩放工具                     |
| R                     | 旋转工具                     |
| E                     | 移动工具                     |
